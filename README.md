# Poetry Maker

Interactive word ripple poetry creation tool for classroom use. Built with Next.js 14, Anthropic Claude API, and Tailwind CSS.

## Features

- **Interactive Word Ripples**: Click on words to create beautiful ripple effects
- **AI-Powered Suggestions**: Get creative word alternatives powered by Claude
- **Poem Generation**: Generate complete poems from prompts
- **Word Analysis**: See word frequency distribution
- **Easy Sharing**: Copy or download your poems

## Setup

### Prerequisites

- Node.js 18+
- npm or yarn
- Anthropic API key
